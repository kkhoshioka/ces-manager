# スマホでシステムを使用する方法（初心者向けガイド）

このシステムを、同じWi-Fiに接続していないスマートフォンや、外出先からインターネット経由で使用するための手順です。
無料のツール「ngrok（エングロック）」を使用するのが最も簡単で安定的です。

## 手順 1: アプリを起動する

まず、パソコン上でシステムを「本番モード」で起動します。これによって動作が軽くなり、エラーが起きにくくなります。

1.  パソコンでターミナル（PowerShell または コマンドプロンプト）を開きます。
2.  以下のコマンドを入力してエンターキーを押します（ビルドと起動を一括で行います）。

    ```bash
    npm run build
    npm start
    ```

3.  「Server running on http://localhost:3000」と表示されたら、システムは正常に動いています。**この画面は閉じないでください。**

---

## 手順 2: インターネット公開用ツール (ngrok) の準備

システムをインターネットからアクセスできるようにする「トンネル」を作ります。

1.  **ngrokのダウンロード**:
    *   公式サイト [https://ngrok.com/download](https://ngrok.com/download) にアクセスします。
    *   Windows版をダウンロードし、解凍（展開）します。
2.  **アカウント作成 (初回のみ)**:
    *   [ngrok.com](https://ngrok.com/) で無料アカウント登録（Sign Up）をします。
    *   管理画面に出てくる「Connect your account」というコマンド（`ngrok config add-authtoken ...`）をコピーして、ターミナルで一度だけ実行します。
3.  **公開コマンドの実行**:
    *   **新しい** ターミナルウィンドウを開きます（サーバーが動いている画面とは別にもう一つ開きます）。
    *   以下のコマンドを入力してエンターキーを押します。

    ```bash
    ngrok http 3000
    ```

---

## 手順 3: スマホでアクセスする

1.  `ngrok` の画面に、以下のようなURLが表示されます。

    ```
    Forwarding      https://xxxx-xxxx-xxxx.ngrok-free.app -> http://localhost:3000
    ```

2.  この `https://...` で始まるURLを、スマホのブラウザ（SafariやChrome）に入力して開いてください。
3.  システムのログイン画面（またはトップページ）が表示されれば成功です！

## 注意点

*   **URLは毎回変わります**: ngrokを停止して再起動すると、URLの `xxxx` の部分は変わります。使うたびに新しいURLを確認してください。
*   **パソコンはつけっぱなし**: パソコンの電源を切ったり、ターミナルを閉じると、スマホからもアクセスできなくなります。
